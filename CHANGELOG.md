# Changelog

AGU follows Semantic Versioning while public contracts stabilize. During 0.x,
breaking changes may occur only in minor releases and must include migration
notes. Patch releases remain backward compatible.

## 0.1.0 - 2026-07-13

- Added installable `agu-basketball` package metadata and `agu` CLI.
- Added typed pipeline foundation and versioned output manifest.
- Added plugin discovery, capability metadata, and diagnostics.
- Added public contract benchmark, license/SBOM governance, CI, and community
  release artifacts.
- Preserved the existing FastAPI, CLI, v3 preprocessing, scoreboard, and player
  identity behavior.
